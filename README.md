# Blended Learning Tool

## A collaborative platform, encouraging gamification and building habits!

## Overview

This is a web application that aims to bridge the gap between games and studying. By combining ideas from other study tools, as well as gamification, the result is a platform inspiring community building and improving one another. Features include but not limited to:
- Unlockable achievements and themes
- Levelling system
- Friend System (add, ignore, etc)
- Leaderboard
- Statistics (keeping track of your goals)
- Flashcards with history

### Technologies Used:
- React (Vite)
- Supabase (Auth & Database)
